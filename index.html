Política de Privacidad

Desarrollador: Orlando Sanchez

Aplicación: Linktek Field

Esta Política de Privacidad aplica a la aplicación Linktek Field y describe cómo manejamos su información.

Nuestra aplicación está dirigida exclusivamente a usuarios mayores de edad. No recopilamos datos de menores de 18 años.

1. Información que Recopilamos y Usamos

a) Datos de Autenticación:

  • Cambio de Contraseña: La aplicación permite cambiar contraseñas existentes. Las contraseñas se almacenan encriptadas.

b) Datos de Localización (Asistencia):

  • Coordenadas GPS: Obtenemos su ubicación exacta (coordenadas) únicamente cuando marca asistencia. Esto nos permite verificar que se encuentra en el lugar de trabajo autorizado.
  • Uso específico: Las coordenadas solo se usan para validar la asistencia laboral y no se comparten con terceros.

c) Funcionalidades de la Aplicación:

  • Gestión de Órdenes: Acceso a órdenes de trabajo del día.
  • Inventario: Consulta y gestión de stock de materiales.
  • Datos Personales: Visualización de información laboral personal.

2. Lo que NO Hacemos

  • ❌ NO creamos cuentas de usuario en la aplicación
  • ❌ NO recopilamos datos personales sensibles adicionales
  • ❌ NO compartimos datos con terceros para marketing
  • ❌ NO usamos la ubicación en segundo plano
  • ❌ NO vendemos datos a ninguna empresa

3. Almacenamiento de Datos

Todos los datos se almacenan de forma segura en nuestros servidores privados. Las contraseñas están encriptadas y las coordenadas de ubicación se utilizan únicamente para validar asistencia.

4. Seguridad

Implementamos medidas de seguridad técnicas para proteger su información. El acceso a los datos está restringido al personal autorizado.

5. Sus Derechos

Usted tiene derecho a:

  • Consultar qué datos tenemos sobre usted
  • Solicitar la corrección de datos incorrectos
  • Plantear preguntas sobre el manejo de sus datos
